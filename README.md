# 💻 Digital VLSI Lab 

This repository contains all Digital VLSI lab experiments performed as part of my **B.Tech ICT curriculum at Marwadi University**.

The experiments focus on **CMOS design, MOSFET characteristics, combinational and sequential circuits, and layout design using Microwind and LTSpice**.

---

## 📌 Objectives

* Understand MOSFET fabrication and operation
* Analyze CMOS inverter characteristics
* Design combinational logic circuits using CMOS
* Implement sequential circuits (Flip-Flops, Latches)
* Perform circuit simulation using LTSpice
* Design layouts using Microwind
* Analyze power consumption and switching behavior

2. Design layouts using **Microwind**
3. Run simulation and observe waveforms
4. Analyze output for correctness
